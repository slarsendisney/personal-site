# Uses

#### Inspired by [Wes Bos](https://wesbos.com/) and [Duncan Bain's](https://duncanbain.dev/) website, here's my config, my tools, my daily carry and development stack!

<h4>Editor and Terminal </h4>

- [Visual Studio Code](https://code.visualstudio.com/) - Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS.
  - Theme: Cobalt2 by [Wes Bos](https://wesbos.com/)
  - Extensions:
    - [Bracket Pair Colorizer 2](https://marketplace.visualstudio.com/items?itemName=CoenraadS.bracket-pair-colorizer-2)
    - [Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)
    - [Prettier: Code Formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
    - [Relative Path](https://marketplace.visualstudio.com/items?itemName=jakob101.RelativePath)
- [Starship Terminal](https://starship.rs/) - The minimal, blazing-fast, and infinitely customizable prompt for any shell!

<h4> Browsers</h4>

- [Google Chrome](https://www.google.com/chrome/)
  - Extensions:
    - [Wappalyzer](https://chrome.google.com/webstore/detail/wappalyzer/gppongmhjkpfnbhagpmjfkannfbllamg)
    - [React Developer Tools](https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi)
    - [svg-grabber](https://chrome.google.com/webstore/detail/svg-grabber-get-all-the-s/ndakggdliegnegeclmfgodmgemdokdmg)
- [PolyPane](https://polypane.app/) - Design debugging and testing, with live reloading, accessibility testing and more.

<h4> Design Tools</h4>

- [Sketch](https://code.visualstudio.com/) - Create, prototype, collaborate, and bring your ideas to life with this all in one tool.
  - Plugins:
    - [Magic Mirror 3](https://magicsketch.io/mirror/)
    - [Sketch Isometric](https://github.com/sureskumar/sketch-isometric)
- [Adobe Creative Cloud](https://www.adobe.com/uk/creativecloud.html) - Photoshop, Lightroom, Illustrator & After Effects
- [SuperPosition](https://superposition.design/) - Extract design tokens from websites and use them in code and in your design tool.
- [Coolors](https://coolors.co/) - Create the perfect palette or get inspired by thousands of beautiful color schemes.

<h4> Design Inspiration</h4>

- [Dribbble](https://dribbble.com/) - Arguably the home of designers online.
